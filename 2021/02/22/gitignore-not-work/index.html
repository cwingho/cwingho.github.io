.gitignore 不生效的原因和解決方法

發表於 2021-02-22 更新於 2022-02-02 分類於 筆記 閱讀次數： Disqus： 本文字數： 229 閱讀時長 ≈ 1 分鍾

有時候把想忽略掉的文件寫入 .gitignore 後，發現沒有生效，那些文件依然會被追蹤。教你 3 句 command 解決問題!

原因

.gitignore 只會對沒有出現在 repository 裡的文件生效。 對已經 push 過的文件會無效。

解決方法

把想忽略的文件從本地 repository 裡刪掉。

懶人包

*記得要先 commit change。
